# 🏡 House Price Prediction using Linear Regression

This project analyzes the relationship between crime rates and median house prices using **Linear Regression**. The objective is to predict house prices based on crime rate data and evaluate how crime impacts property values.

---

## 📌 Project Overview
- Uses **Linear Regression** to model the relationship between crime rate and house prices.
- Implements **data visualization** to explore the dataset.
- Evaluates the model using **Mean Squared Error (MSE)** and **R² score**.
- Provides insights on how crime rate influences housing prices.

---

## 📂 Files & Directories
- `houseprice.ipynb` - Jupyter Notebook with the full analysis and model implementation.
- `requirements.txt` - List of dependencies needed to run the project.
- `data/` - Directory for dataset files (if applicable).
- `README.md` - Project documentation.
